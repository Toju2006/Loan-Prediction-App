# Loan Prediction App 🚀

A machine learning web app that predicts loan approval based on user inputs.

## 📌 Overview
This project uses a trained Random Forest model to predict whether a loan application will be approved or rejected.

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Features
- User-friendly web interface
- Real-time predictions
- Trained ML model (Random Forest)

## 🚀 How to Run
1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## 📁 Files
- `app.py` → Streamlit app
- `loan_model.pkl` → Trained model
- `loan_data.ipynb` → Model training notebook
